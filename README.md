# BankSight-Transaction-Intelligence-Dashboard
BankSight is a financial analytics system built using Python,Streamlit and SQLite3. It allows user to  explore Customer,Account,Transaction,Loan and Support Data,perform CRUD operations,simulate  deposits/withdrawls, and view analytic insights.

Objectives :
• Understand customer & transactions behaviour.

• Detect anomalies and potential fraud.

• Enable CRUD operations on all datasets.

• Simulate banking transactions (Credit/Debit).
